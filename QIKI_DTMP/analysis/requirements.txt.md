## Примеры использования
1. ```bash
python -m venv venv && source venv/bin/activate
```
2. ```bash
pip install -r requirements.txt
```
3. ```bash
pip list --outdated
```
4. ```bash
pip install --upgrade protobuf
```
5. ```bash
pip freeze > requirements.lock
```
